# Installation

1. `git clone matthew@git.komputerwiz.net:git/dotfiles.git ~/.dotfiles`
2. `cd ~/.dotfiles`
3. `stow <package>`

Some packages have subpackages (prefixed with `@`). See package README for
details and instructions.

# Updating

1. `cd ~/.dotfiles`
2. `git pull`
